## 🚫 禁止事項 / 免責事項 (Prohibited Matters)

本リポジトリのソースコードおよび成果物は、東京国際工科専門職大学の講義（devpro3）における第4班（G4）のグループワーク成果物です。
グループの構成員（メンバー）以外の利用において、以下の行為を固く禁じます。

- **丸コピ（盗用）の禁止:**
  ソースコードをそのまま、あるいは一部を改変して自身や自身のグループの課題や成果物として提出する行為。
- **無断引用・参考の禁止:**
  本リポジトリのコードを引用して自身のプロジェクトに組み込む行為（事前に開発メンバーの許可を得た場合を除く）。
- **二次配布・商用利用の禁止:**
  本リポジトリのコードを他者へ共有・転載する行為、および商業目的に利用する行為。

### 🚨 違反した場合の対応
万が一、学内等での無断転載や丸コピ（ソースコードの一致）やApp内や、ライブアクティビティや、ウィジェットのSwiftUI以外のデザインの一致が発覚した場合は、**担当教員への報告を含めた適切な措置**を取らせていただきます。
